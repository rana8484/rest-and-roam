Rest And Roam

Android App — Java, SQLite, REST APIs

Overview
Rest and Roam is an Android travel and accommodation app that allows users to explore rental listings, village-based activities, restaurants, and cafés.

Features
Browse rentals, restaurants, and activities
Village-specific search filters
Full offline support using embedded SQLite
REST API integration for remote data sync

Tech Stack

Mobile: Java, Android Studio
Database: SQLite (offline-first)
Backend: REST APIs
